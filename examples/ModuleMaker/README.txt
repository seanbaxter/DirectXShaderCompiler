//===----------------------------------------------------------------------===//
//                         ModuleMaker Sample project
//===----------------------------------------------------------------------===//

This project is an extremely simple example of using some simple pieces of the 
LLVM37 API.  The actual executable generated by this project simply emits an 
LLVM37 bitcode file to standard output.  It is designed to show some basic 
usage of LLVM37 APIs, and how to link to LLVM37 libraries.
